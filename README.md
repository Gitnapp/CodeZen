# code zen

> A calm, distraction-free Visual Studio Code theme

![Version](https://img.shields.io/badge/version-0.0.8-blue)
![VS Code](https://img.shields.io/badge/VS%20Code-%E2%89%A5%201.0.0-orange)

## Features

- **Deep dark background** - Easy on the eyes for long coding sessions
- **Soft, muted colors** - Reduced eye strain with carefully selected accent colors
- **Zen-inspired palette** - Purple, blue, green accents with neutral grays
- **Consistent syntax highlighting** - Support for 50+ languages with balanced color distribution
- **Full UI customization** - Styled scrollbars, minimap, tabs, menus, and status bar

## Installation

### From VS Code Marketplace
1. Open VS Code
2. Go to Extensions (`Cmd+Shift+P` → "Extensions")
3. Search for `code zen`
4. Click Install

### From Release
1. Download the `.vsix` file from [Releases](https://github.com/Gitnapp/code-zen/releases)
2. Open VS Code
3. Go to Extensions (`Cmd+Shift+P` → "Extensions")
4. Click "Install from VSIX"

## Activation

1. Go to `Code` → `Preferences` → `Color Theme` (or `Cmd+K` `Cmd+T`)
2. Select `code zen`

## Supported Languages

JavaScript, TypeScript, Python, HTML, CSS, JSON, Markdown, Go, Rust, Java, C/C++, Ruby, PHP, SQL, Shell, and more.
